## ğŸ¯ Usage

### Adding Transactions

1. Click **"Add Transaction"** button
2. Fill in amount, category, description, and date
3. Select type (Income/Expense)
4. Click **"Add"** to save

### Importing CSV

1. Click **"Import"** button
2. Select CSV file with columns: Date, Amount, Description, Category
3. App automatically detects and categorizes transactions
4. Review imported data

### Setting Budgets

1. Navigate to **"Budgets"** tab
2. Click **"Add Budget"**
3. Select category and set limit
4. Choose period (monthly/weekly)
5. Track spending in real-time

### Creating Savings Goals

1. Navigate to **"Savings Goals"** tab
2. Click **"Add Goal"**
3. Enter goal name, target amount, and deadline
4. Choose custom color
5. Add funds incrementally

### Using AI Assistant

1. Click **"AI Chat"** button
2. Ask questions like:
   - "Analyze my spending"
   - "How can I save more?"
   - "Budget recommendations"
   - "Financial health check"
3. Get instant, personalized insights
